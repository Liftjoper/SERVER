# Fastify-сервер с простым интерфейсом

Небольшой проект на Node.js с использованием фреймворка Fastify.  
На главной странице — кнопка, при нажатии отправляется запрос на `/api`, сервер отвечает сообщением, а в консоли браузера появляется соответствующая надпись.

## Как получить проект

Склонируй репозиторий (если у тебя есть доступ) или скачай архив с кодом:


git clone  https://github.com/Liftjoper/SERVER
cd SERVER


Без Git:
Нажми зелёную кнопку «Code» → «Download ZIP». Распакуй архив в удобную папку. Затем открой терминал и перейди в эту папку командой cd путь/к/папке.

## Что нужно для запуска

- Установленный Node.js (версия 14 или выше).
- Скачанные файлы проекта (можно склонировать репозиторий или просто забрать архив).

## Как развернуть

1. Открой терминал в папке проекта.
2. Выполни команду `npm install`, чтобы подтянуть зависимости (fastify и плагин для статики).
3. Запусти сервер: `npm start` или `node server.js`.
4. В консоли сервера увидишь сообщение, что он слушает порт 3000.

## Проверка работы

- Открой браузер и перейди на `http://localhost:3000`.
- Нажми на кнопку «Сделать запрос на АПИ».
- Открой инструменты разработчика (F12) и посмотри в консоль — там должно появиться «Запрос прошел успешно».

Если сервер не запускается, проверь, не занят ли порт 3000 другим приложением, или попробуй изменить порт в файле `server.js`.

## Остановка сервера
В терминале, где запущен сервер, нажми Ctrl+C (два раза, если нужно). Сервер остановится.

## Файлы в проекте

- `server.js` — сам сервер.
- `public/index.html` — страничка с кнопкой.
- `package.json` — список зависимостей и скрипты.
- `README.md` — этот файл.
